# Nemous plugin marketplace

Claude Code plugins by [Nemous](https://github.com/nemous-x).

## Add the marketplace

```
/plugin marketplace add nemous-x/nemous
```

## Plugins

| Plugin | Description | Install |
|---|---|---|
| [F67](https://github.com/nemous-x/F67) | AI Development Operating System — Domain Driven Memory, specialized agents, deterministic lifecycle workflows | `/plugin install f67@nemous` |
| [NemoHire](https://github.com/nemous-x/nemohire) | AI-powered hiring copilot — professional identity, job sourcing/ranking, tailored applications, sequential apply automation, application tracker, email sync. Works in Claude Code and Cowork. | `/plugin install nemohire@nemous` |

New plugins are added as entries in `.claude-plugin/marketplace.json`, each sourced from its own repository.
